The ProR Platform for Requirements Engineering and Formal Modeling

ProR is a tool for requirements engineering that supports the ReqIF 1.0.1 Standard natively. It is Eclipse-based and is built for extensibility.

Please visit the ProR page at Eclipse for the latest information.

ProR is part of the Requirements Modeling Framework (RMF).

ProR and RMF are still in Incubation and currently under active development. Please use with care.
