OMG ReqIF XML Schema

The plug-in is accompanied by the OMG ReqIF XML Schema files. The OMG ReqIF specification is available at: http://www.omg.org/spec/ReqIF/.

The following OMG ReqIF XML Schema files are located in the plug-in:

  • schema/reqif.xsd obtained from http://www.omg.org/spec/ReqIF/20110401/reqif.xsd
  • schema/driver.xsd obtained from http://www.omg.org/spec/ReqIF/20110402/driver.xsd

Your use of the OMG ReqIF XML Schema files is governed by the terms and conditions of the Object Management Group License a copy of which is included in the file about_files/OMG_NoticeAndLicense.html and is also available in the ReqIF specification http://www.omg.org/spec/ReqIF/1.1/PDF.

W3C XML namespace Schema

The plug-in is accompanied by XML Namespace XML Schema file which are referenced by the OMG ReqIF Schemas and the W3C XHTML Modularization Schemas. The XML Namespace specification is available at: http://www.w3.org/XML/1998/namespace.html.

The following XML namespace XML Schema file is located in the plug-in:

  • schema/xml.xsd obtained from http://www.w3.org/2001/xml.xsd
