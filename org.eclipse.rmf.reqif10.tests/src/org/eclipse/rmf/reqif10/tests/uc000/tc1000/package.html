TC0001000 "Containment EStructuralFeature"

Checks whether containment references are handled correctly

The following test goals are addressed

ID isReference isContainment isMany isSet isDefaultValue isNull ReqIF - Feature
1 x x - - isDefaultValue isNull ReqIF - Feature
  1. 1 feature with isMany=false, unset
  2. 2 feature with isMany=false, set, primitive, default value
  3. 3 feature with isMany=false, set, primitive, non-default value
  4. 4 feature with isMany=false, set, complex, null
  5. 5 feature with isMany=false, set, complex, non-null
  6. 6 feature with isMany=true, unset
  7. 7 feature with isMany=true, set, null
  8. 8 feature with isMany=true, set, empty
  9. 9 feature with isMany=true, set, non empty (size=1)
  10. 10 feature with isMany=true, set, non empty (size=2)
Note: it is not required that the serialized artifacts are valid with respect to the XML schema. E.g. it is allowed that mandatory elements are not serialized or that the root element is not the reqif element.
